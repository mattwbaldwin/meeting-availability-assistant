# Meeting Availability Assistant

A lightweight Chrome extension that helps you propose meeting times directly from Gmail.

## What it does
- Detects availability requests in emails
- Checks your calendar for open times
- Lets you select the times you want to propose
- Inserts a clean, human-readable reply into your draft

## Privacy
This extension does not collect or transmit personal data.
See [PRIVACY.md](PRIVACY.md) for details.
